    Outils

    Auto Codebase Documenter - Pour documenter automatiquement votre code

    Auto Codebase Documenter

    Outil python qui évalue et documente automatiquement le code en utilisant la puissance de GPT-3.5-turbo ou GPT-4. L'outil nécessite une clé OpenAI pour pouvoir être utilisé.

    Extensions VS Code

    Console Ninja

    console.ninja.com VS Code marketplace

    Une extension qui affiche les sorties console.log et les erreurs d'exécution directement dans l'éditeur.

    Image preview

    VS Code marketplace

    Vous permet de prévisualiser les images importées dans votre code dans l'éditeur.

    Import Cost

    VS Code marketplace

    L'extension affiche dans l'éditeur la taille du package importé.

    Path Intellisense

    VS Code marketplace

    Ajoute de l'auto-complétion pour les fichiers.

    Une extension pour les mesures, les informations et le suivi du temps.
