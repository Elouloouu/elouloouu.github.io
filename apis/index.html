    APIs

    Leaf

    withleaf.io

    Une API pour la nourriture et l'agriculture qui permet de récupérer les données agricoles.

    Spoonacular

    spoonacular.com

    Vous donne accès à plus de 330 000 recettes et 80 000 produits alimentaires.

    Spotify

    developer.spotify.com/

    L'API de spotify vous permet de récupérer des données sur des artistes, de la musiques, des albums, des titres directement depuis le catalogue de Spotify.

    NewsAPI

    newsapi.org

    Chercher les actualités parmis plus de 30 000 publications avec cet API gratuit pour les projets non commerciaux.

    Pexels

    www.pexels.com

    Intégrez des milliers de photos et videos directement dans vos produits.

    Fixer

    fixer.io

    API simple et légère poru récupérer les taux de change historiques et actuels.

    Clarifai

    clarifai.com

    Cette API reconnaît plus de 11 000 concepts comme des chiens, des chats etc ...

    Quickchart

    quickchart.io

    Permet d'ajouter des graphiques dans votre projet.

    Coinbase

    coinbase.com

    Utile pour récupérer le prix des crypo-monnaies, pour les acheter et plus encore ...

    Cloud Natural Language

    cloud.google.com

    Cette API de Google fait partie des nombreuses API de Google de Machine Learning.

    Nasa

    api.nasa.gov

    Vous donne accès à différents types de données à propos de l'astronomie, la météo spatiale, la Lune, Mars et plus encore ...

    MovieDB

    themoviedb.org

    Permet des récupérer des données sur des films, des acteurs, des images, des séries etc ...

    Count

    countapi.xyz

    Vous donne la possibilité de créer des compteurs numériques pour tracer le nombre de fois qu'une page a été visitée.

    Yahoo Finance

    financeapi

    Une API qui aide à récupérer des données financières (actions, cotations ...)

    TempMail

    temp-mail.org

    Une API qui permet d'obtenir une adresse email anonyme qui s'auto-détruit après un certain temps.
